# Shadow HT - Application Android

Cette branche contient la version Android de Shadow HT.

## Installation et Configuration

### Prérequis
- Android Studio (dernière version)
- JDK 11 ou supérieur
- SDK Android 21+

### Étapes d'installation

1. Clonez la branche android-app :
   ```bash
   git clone -b android-app https://github.com/jeanpierrewoodson13-web/Shadow-ht.git
   ```

2. Ouvrez le projet dans Android Studio

3. Synchronisez les dépendances Gradle

4. Connectez un appareil Android ou lancez l'émulateur

5. Cliquez sur "Run" pour construire et installer l'application

## Caractéristiques

- ✅ WebView intégrant votre site web
- ✅ Support JavaScript et stockage de données
- ✅ Navigation arrière (bouton retour Android)
- ✅ Connexion Internet requise
